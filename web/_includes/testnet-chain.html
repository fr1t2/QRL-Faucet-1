{{site.coin}} Testnet

Bootstrap Files

Use these files to speed up the syncing time for a {{ site.coin }} testnet node.

The most secure approach is to spend the time to fully sync your node from scratch. This ensures your local chain is accurate to all other modes and that you are not running a forked chain.
If you don't need this security, you can use these files to bootstrap your node setup.

Un-tar these state files into your local ~/.qrl/data/ folder and start the node. This will start at the latest blockheight these files were made and only sync from there.
