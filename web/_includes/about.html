Open and Secure

Building open source and free tools to interact with the Zond POS blockchain. Using advanced cryptography and quantum resistant systems to secure the future. Dedicated to bringing innovation and future-minded thinking to the masses with easy to digest systems and tools built off the Zond blockchain project.


Strong Cryptography

Zond uses NIST Approved quantum resistant cryptography for various functions allows the use of stateful XMSS trees. We aim to bring this cryptography into an easy to use public interface with a focus on security and privacy.
